### 影武者　　
影武者はTeamsで退席中ステータスになるのを防ぐアプリです。  
お茶買いに行ったりするときに使います。  
tauri触ってrustの勉強しようと思って作成しましたが、ほとんどweb技術でできてしまいました。。。

![実行中の様子](docs/img/通常画面.png)

・やってることは1分間ごとにwebviewからtauriコマンドを呼んであげる。  
・tauriコマンドでctrlキーを押下する。  

です。
